## Getting Started

First, setup the environment file first.

```bash
# create .env file based from .env.example
cp .env.example .env

# set necessary values
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
